## 🚀 Local Development

To run the website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/narayanaconsultingservicesllp/narayanaconsultingservicesllp.github.io.git
   cd narayanaconsultingservicesllp.github.io
   ```

2. Start a local web server:
   ```bash
   # Using Python 3
   python3 -m http.server 8000
   
   # Or using PHP
   php -S localhost:8000
   
   # Or using Node.js (with http-server)
   npx http-server -p 8000
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```

## 🌐 Deployment

This site is hosted on GitHub Pages and automatically deploys from the `main` branch.

### Deployment Steps:

1. Push your changes to the `main` branch:
   ```bash
   git add .
   git commit -m "Update website"
   git push origin main
   ```

2. GitHub Pages will automatically build and deploy the site

3. The site will be available at:
   ```
   https://narayanaconsultingservicesllp.github.io
   ```

### GitHub Pages Configuration:

1. Go to repository Settings → Pages
2. Source: Deploy from branch `main`
3. Folder: `/ (root)`
4. Save the configuration
